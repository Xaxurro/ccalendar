# ccalendar
Utility to set reminder of events written in C++!

# How to compile
```
git clone git@github.com:Xaxurro/ccalendar.git
cd ccalendar
make release
```

## OPTIONAL: Move it into your `bin` directory to use it from everywhere
`sudo mv ccalendar /bin`
